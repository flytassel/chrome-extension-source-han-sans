# 强制思源黑体 Chrome 扩展

这是一个轻量级的 Chrome 扩展，旨在强制网页使用**思源黑体**，替换掉如宋体等默认字体。

## 功能特点
- 仅在白名单网站上强制替换所有元素的字体。
- 优先使用思源黑体及其他高质量无衬线字体。
- 在 `document_start` 阶段注入样式，确保页面加载时即应用。
- **白名单模式**：默认不启用。您可以通过弹出菜单手动为特定网站开启。

## 如何为网站开启
1. 访问您想要应用思源黑体的网站。
2. 点击扩展图标。
3. 点击“**添加到白名单 (启用)**”。页面将自动刷新并应用新字体。

## 如何安装
1. 下载或克隆本仓库。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的“**开发者模式**”。
4. 点击“**加载已解压的扩展程序**”，选择包含本扩展的目录。
5. 刷新网页即可生效。
